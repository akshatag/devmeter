# DevMeter - Developer Scoring App

DevMeter is a web application that analyzes your GitHub activity and provides you insights into your performance as well as a holistic score.

## Core Features

- **Login with Github**
- **View your insights**
- **View your score**   
- **Share your scorecard** 

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- MongoDB
- GitHub API
- React Query
- Shadcn UI
- Jest
